Spec — Home

Route /
Komponenten: TopNav (≥768) / BottomNav (<768) · SearchBar · HeuteCard-Karussell · FratcherTeaser · Section „Entdecken" (klickbare Überschrift) · RecipeCard-Grid · InfiniteScrollSentinel.
States: Loading = 6 Card-Skeletons + Hero-Skeleton. Empty = Feed nie leer (kuratiert); falls doch → „Noch nichts zu entdecken". Error = Inline-Banner „Konnte nicht laden — Erneut versuchen" über dem Grid; Hero fällt auf statisches HeuteCard zurück.
Infinite Scroll: IntersectionObserver auf Sentinel (rootMargin 600px) → nächste Seite (page-size 12) via GET /api/recipes?page=n (zu prüfen). Spinner während Fetch, „Alles gesehen" am Ende. Kein Fetch im Error-Zustand.
Responsive: Grid 2→3 ab 768px, zentrierter Container max 1120px; Hero 16/9; TopNav ersetzt BottomNav ab 768px.
Datenbindung: Hero ← featured[] (Titel, image_url, id) · Feed ← recipes[] → RecipeCard.
Links: „Entdecken" + Chevron → /recipes · Fratcher-Teaser → /fratcher · Hero/Cards → /recipes/:id.
a11y: SearchBar sichtbarer Fokus-Ring (Token-Accent); Karussell-Dots mit aria-label; Herz aria-pressed; Kontrast Hero-Text über Verlaufsmaske gesichert.
