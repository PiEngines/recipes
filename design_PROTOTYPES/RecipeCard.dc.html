Spec — RecipeCard

Komponente <RecipeCard recipe={…} />
Props / Datenbindung: image_url|null → Medienfläche (Fallback: Terracotta→Olive-Gradient) · title → H3 (line-clamp:2) · author.name → Meta · total_time → Meta · is_favorite → Herz-Zustand · status==='pending' → „Wird geprüft"-Badge (Feldname zu prüfen).
States: Loading = Skeleton (Medienfläche + 2 Textzeilen als Shimmer). Empty = n/a (Card ist nie leer; leerer Grid wird von der Seite behandelt). Error-Bild = Gradient-Fallback + Alt-Text.
Responsive: Card selbst fluid (füllt Grid-Spalte); Medien-aspect-ratio:4/3 fix. Grid-Spalten setzt die Elternseite (2→3 ab 768px).
Links: ganze Card → /recipes/:id. Herz-Button = eigener Klickbereich, stopPropagation, kein Navigieren.
Dark Mode: ausschließlich Tokens. Herz-Umriss --heart-outline (dark = transparent → nur Füllung sichtbar).
a11y: Card = <a> mit Titel als Accessible Name; Herz-Button aria-pressed + aria-label; Badge zusätzlich als Text (nicht nur Farbe); Medien-Alt = Titel.
